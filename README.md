# Vrinda Store Data Analysis (Excel Project)

## Overview
This project analyzes sales data of Vrinda Store using Excel.

## Key Insights
- Monthly sales trend
- Orders vs Sales comparison
- Sales by gender
- Top 5 states by sales
- Order status distribution
- Channel-wise orders

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## Files
- Vrinda_Store_Analysis.xlsx

## Outcome
Built an interactive dashboard to derive business insights and visualize sales performance.
